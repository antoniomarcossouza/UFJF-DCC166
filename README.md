# TVC 1 DCC166

Sistema de Apoio à Decisão para tratamento da Doença de Gaucher no SUS, com dados do SIA (base `AM`) via PySUS, materialização em Parquet e dashboard Streamlit com análise de sensibilidade.

URL do dashboard final: https://dcc166-tvc1.streamlit.app

## Referências
- [PySUS - SIA](https://pysus.readthedocs.io/en/latest/databases/SIA.html)
- Borin et al. (2024). *Gaucher disease in Brazil*. Front. Pharmacol. [DOI 10.3389/fphar.2024.1433970](https://doi.org/10.3389/fphar.2024.1433970)
